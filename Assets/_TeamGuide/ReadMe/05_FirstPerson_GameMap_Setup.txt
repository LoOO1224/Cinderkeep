Cinderkeep First Person Game Map Setup

목적
- !! Play Our Game 버튼이 여는 Cinderkeep_Game 씬에서 1인칭 플레이어 스크립트를 바로 테스트하기 위한 준비입니다.
- 아직 플레이어 담당 스크립트가 없으므로, Player 오브젝트와 테스트 맵만 먼저 준비했습니다.

씬 위치
- Assets/Scenes/MainGame/Cinderkeep_Game.unity

플레이어 스크립트 붙일 오브젝트
- Player

현재 준비된 컴포넌트
- CharacterController
- 하위 Transform_CameraRoot_FirstPerson
- 하위 Camera_FirstPerson

플레이어 스크립트 연결 기준
- PlayerMovement, PlayerStatus는 Player에 붙입니다.
- PlayerView가 카메라 회전을 담당한다면 Transform_CameraRoot_FirstPerson 또는 Player에 붙입니다.
- HUDView가 필요하면 Canvas_GameHUD_Test 아래 Panel_HUDRoot에 붙입니다.
- Scene View에서는 Player 캡슐을 볼 수 있고, Game View에서는 Camera_FirstPerson으로 1인칭 확인을 합니다.

주의
- 기존 1인칭 플레이어/카메라 프로토타입은 테스트 혼선을 막기 위해 제거했습니다.
- 예전 Canvas_Cinderkeep_HUD는 테스트 혼선을 막기 위해 제거했습니다.
- 테스트 카메라를 우선 사용하도록 기존 카메라와 AudioListener는 비활성화했습니다.
- 새 플레이어 스크립트가 오면 먼저 WASD 이동, 마우스 회전, 점프, 달리기만 확인합니다.
